```bash
cd AutoMPP

# Install dependencies
npm install
```

## 🚀 Usage

```bash
# Run AutoMPP
npm start
```

Or customize your strategy:

```javascript
import { runStrategy } from "./src/core/engine";

runStrategy({
  strategy: "auto",
  riskLevel: "medium",
  assets: ["BTC", "ETH"],
});
```
